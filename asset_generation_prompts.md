# KaushalAI - AI Image & Video Asset Prompts

Kyunki website ko ab aur bhi zyada professional banana hai, hum **icons/logos ki jagah real images aur video frames** use karenge. Jab user site par aayega, toh ek premium video loop chalega jo site ko poori tarah se "$10,000 SaaS" startup jaisa feel dega.

Aap in prompts ko **Midjourney, RunwayML, Luma, Pika, Kling, Whisk** ya kisi bhi AI generator mein daal sakte ho. Inki language bilkul waise hi likhi hai jo AI engines easily samajh sakein.

### Pro-Tips:
- **Video Slider:** In videos ko loop mein chalane ke liye MP4 format me export karke CSS mein `pointer-events-none`, `muted`, `autoplay`, aur `loop` lagana. Responsive design ke liye `object-fit: cover` ka use karein.
- **Color Consistency:** Hamesha "dark luxury", "neon cyan/purple", aur "glassmorphism" ka use karein taaki website ki theme ke saath perfectly sync karein.
- **Interactivity:** Whisk/NanoBanana ka use interactive icons ke liye karein (PNG/WEBP blocks with transparency).
